It first came into effect in 1996 with CSS1, which introduced basic features for text styling, font properties, margins, padding, color and background styling, and the box model. Basic Syntax Rules:

In CSS, a rule consists of a selector that targets HTML elements, and a declaration block containing properties and values. Example:

p {
    color: blue;
    font-size: 16px;
}
Try it yourself

At-Rules (@):

At-rules are special CSS statements that start with @ and control CSS behavior.

Some examples: [17]

  • @import: Allows you to import a style sheet into another style sheet.
  • @keyframes: Controls the steps in an animation by defining styles for points along the animation sequence
  • @font-face: Specifies a custom font with which to display text
  • @media: Applies parts of a style sheet based on the result of one or more media queries
  • @page: Customizes the dimension, orientation, and margins of printed pages

Comments:

Used to add notes inside CSS, ignored by browsers:

Selectors:

Selectors define which HTML elements the CSS rules apply to: [18]

  • Type Selector: ex: p
  • Class Selector: ex: .classname
  • ID Selector: ex: #idname
  • Pseudo-class Selector: ex: a:hover
  • Pseudo-element Selector: ex: p::first-letter

!important Declarations:

Overrides normal cascading rules:

p {
    color: blue !important;
}

One of the most exciting features of CSS are 2D and 3D transforms which let you move, rotate, scale, and skew elements. 2D Transforms work on the X and Y axe and 3D Transforms add depth using the Z axis.These effects create dynamic and visually engaging interfaces without needing complex animations.

To use them, you must use the transform property which has functions like translate()/translate3d(), rotate()/rotate3d(), scale()/scale3d(), taking the dimensions as parameters

Modern CSS adapts automatically to different devices through several key features. Responsive design allows web pages to resize and rearrange elements based on screen width, making them readable and usable on desktops, tablets, and phones without extra code.

Some of these features include:

  • Media Queries:

    Apply styles based on screen characteristics. Example: 

    @media (max-width: 600px) 
    @media (orientation: landscape)

  • Flexible Layouts:

    Organizes content in flexible rows or columns. Example: 

    display: flex;
    display: grid;

  • Fluid Images:

    Scale images according to their container. Example: 

     max-width: 100%; height: auto;
